3. Architecture & Tech Stack

Core Services:

PXE/TFTP/HTTP/DHCP Proxy in Go (fast, portable).

WinPE/Linux boot support via iPXE.

Image Handling:

Windows DISM & wimlib for WIM/FFU.

Partclone/Clonezilla libs for Linux/macOS imaging.

Web Dashboard:

Frontend: React + Tailwind.

Backend: Go (Gin/Fiber) or Rust (Actix).

Auth: JWT + Role-based Access.

Cloud Integration:

S3 SDK (AWS/GCP/Azure).

rclone integration for other storage.

Packaging:

Standalone EXE for Windows.

Docker container for Linux/cloud.

Optional:

Local DB: SQLite/Postgres for deployment history.

Remote Agent for scaling multiple Bootah nodes.
